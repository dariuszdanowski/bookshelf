# catalog-search-and-filters (S-08)

Wyszukiwarka katalogu: pełnotekst (tytuł, autor, wydawnictwo) + filtry kolor grzbietu / półka (multi-select) / status przeczytania, kombinowalne; wyniki pokazują nazwę półki + pozycję + status. Brak wyników → „nie masz tej książki". Domyka momenty in-bookstore (US-03) i in-home (US-04). PRD refs: FR-032–036; NFR p95<1s na ~1000.

Tryb: Fast track. Świadome cięcie: „krótki opis z publicznej bazy" (FR-032) odroczony — nie persystowany, backfill wymaga re-fetch API (follow-up slice).
